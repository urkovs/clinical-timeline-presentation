# VP's Clinical Journey

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/`.

## Editing

Timeline text lives in `script.js` inside the `milestones` array. The visible chart intentionally uses short labels; fuller context appears in the right-side detail panel.

Use the date slider or the Previous/Next buttons to move through the case manually. ArrowLeft and ArrowRight also step between milestones.
